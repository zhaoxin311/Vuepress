### 初始化项目
1. 安装脚手架`tdesign-starter-cli`，在安装之前，确保`node` 与`npm`的开发环境。
```
npm i tdesign-starter-cli
```

2. 创建项目
```
td-starter init
```

### 启动项目
1. 安装依赖
```
npm install
```
2. 运行项目
```
npm run dev
```
tip:如果启动失败，可通过`node -v`检查node版本是否在12.0.0以上。

执行成功之后，打开浏览器，你会拥有一个基于TDesign，提供了内置多种页面和配置的中后台系统。
